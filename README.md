# Personal Curriculum Vitae Repository
## Update in VS Code using LaTeX Workshop
Open main.tex. Then either:
* press Cmd+Option+B
* or click the → Build button in the top-right
The PDF should compile automatically.

Options:
* Cmd+Option+V
* or click “View LaTeX PDF”

You’ll get a live side-by-side PDF preview.
## Push to Git and compile using GitHub actions
* GitHub Action builds PDF automatically
* GitHub repo → Actions → latest run → Artifacts

Compiled PDF will be called Joseph_Mullen_CV_*.pdf as defined in build-pdf.yml
